isoWATFLOOD

This code is developed to run with the hydrologic portion of the WATFLOOD mode, CHARM (Canadian Routing and Hydrologic Model). CHARM is available on GitHub; isoWATFLOOD will not run without CHARM. isoWATFLOOD code is the collaborative effort of Dr. T. Stadnyk and Dr. T. Holmes; developers assume no libability for the code or its output. This code is provided for research purposes. To report issues or for assistance, please contact tricia.stadnyk@ucalgary.ca.

HYPE-HDS

HYPE-HDS is a modified version of the HYPE model that allows HYPE to simulate the variable contributing areas of prairie basins and their effects on the watershed response. The HDS model simulates the change in the contributing area, depressional storage, and outflow from the subbasin.
